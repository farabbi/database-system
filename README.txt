1.已为您提供Makefile文档，请执行make命令进行编译
2.如需重新编译，请先执行make clean，再执行make
3.make clean会同时删除db目录，请确保重要的数据得到了备份
4.请确保db目录存在再运行程序

Special features：
1.程序执行时，其后可附带任意数量的包含SQL语句的文件名作为命令行参数。
若其后未包含文件名，则可通过标准输入输入SQL语句，在该模式下，程序需
通过Ctrl + c结束运行。
2.连接和选择的条件可以任意顺序排列。
3.可同时处理多个聚集。
4.在select结果的末尾会显示行数。
5.输入超出实验要求的query会显示too difficult。
